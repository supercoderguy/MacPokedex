# MacPokedex
Simple Pokédex app for macOS

## Requirements (Python)
`requests 2.31.0`
`tqdm 4.66.1`

## Troubleshooting
If you get an error saying 'MacPokédex Not Opened',
1. Go to settings
2. Scroll down to 'Privacy & Security'
3. Scroll down and click 'Open anyway'
4. Click 'Open anyway' on the pop-up
